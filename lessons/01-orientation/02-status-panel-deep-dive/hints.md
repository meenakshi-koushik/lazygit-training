# Hints

## Hint 1

Start by looking at the panels in lazygit. The Status panel is at the top-left. Notice the branch name and the ahead/behind count relative to origin. To work with branches, you need to navigate to the **Branches** panel -- try pressing `3` or using `h`/`l` and `j`/`k` to get there.

## Hint 2

In the Branches panel you will see all the local branches listed. Use `j`/`k` to move to `feature/auth-service`, then press `<space>` to check it out. After switching, look at the Status panel again -- it should now show `feature/auth-service`.

## Hint 3

Now that you are on `feature/auth-service`, you need to create a new branch from it. While in the Branches panel, press `n` to create a new branch. Type `status-explored` as the branch name and press `<enter>`. Verify the Status panel now shows `status-explored`.
